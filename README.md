# YouTube Video Summarizer - Chrome Extension

## Overview
The **YouTube Video Summarizer** is a Chrome extension that provides quick and concise summaries of YouTube videos using AI-powered text processing. It helps users save time by extracting key points from long videos without needing to watch them entirely.

## Features
- **Instant Summaries**: Generates a summary of any YouTube video with a single click.
- **AI-Powered**: Uses advanced NLP models to extract the most important points.
- **Supports Multiple Languages**: Works with videos in different languages.
- **User-Friendly Interface**: Simple and intuitive UI for seamless interaction.
- **Copy & Save**: Easily copy the summary for later use.

## Installation
1. Download the extension from the Chrome Web Store (or manually install it using developer mode).
2. Click on the extension icon to activate it while watching a YouTube video.
3. Get an instant summary displayed on the screen.

## How to Use
1. Open YouTube and play any video.
2. Click on the **YouTube Video Summarizer** extension icon.
3. The extension will analyze the video and generate a summary.
4. Copy or save the summary for reference.

## Permissions Required
- **Access to YouTube**: To analyze video content and generate summaries.
- **Storage**: To save user preferences and summaries.
